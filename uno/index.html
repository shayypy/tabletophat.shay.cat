UNO

Play UNO with your friends in Discord! This bot ports the classic UNO card game to your Discord server, making use of buttons to keep private, easy-to-use hands without spamming your DMs or a server channel.

Rules

The logic for playing was based on the classic UNO set rules, so you should be familiar already if you've played standard UNO before. However, there are a couple minor alterations, listed here, along with some standard-but-pedantic rules that you might want to brush up on.
